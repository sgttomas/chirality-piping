It is not an engineering design basis, not a standards interpretation, not a
project specification, and not suitable for professional reliance.

OpenPipeStress rule-pack schema surface. It must not be used as a source of
engineering criteria, material properties, stress limits, allowables,
SIF/flexibility data, dimensional tables, owner requirements, vendor data, or
code compliance decisions.

Any real rule pack must be supplied by a responsible user or organization,
reviewed for provenance and redistribution rights, kept private where required,
and evaluated by competent human engineering judgment before use on a real
project.

The software may report user-rule checked, user-rule failed, missing-input, or
human-review-required states. It does not certify, approve, seal, authenticate,
or declare engineering code compliance for reliance.
